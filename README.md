### 3. Hypothesis Testing

**Business Question:**

Is there a significant difference in average sales between the Electronics and Grocery categories?

**Null Hypothesis (H₀):**
There is no significant difference in average sales between Electronics and Grocery.

**Alternative Hypothesis (H₁):**
There is a significant difference in average sales between Electronics and Grocery.

**Statistical Test Used:**

* Independent Two-Sample T-Test

The test was performed using Python and the SciPy library.
